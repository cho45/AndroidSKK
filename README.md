SKK for Android fork
====================

[SKK for Android 3.4](http://ray-mizuki.la.coocan.jp/software/skk_jp.html) からの自分用 fork になります。

変更点
======

 * egg-like-newline な挙動に (変換中の RET で改行を入力しない)
 * 辞書登録時に何も入力せず RET した場合は元の状態に戻るように
